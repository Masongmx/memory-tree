# 🌳 记忆树

你的龙虾会记住每一件事——但这是个问题。过期信息堆积，token 白烧，重要知识被噪音淹没。

记忆树让龙虾学会**遗忘**：常用知识常绿，不用自动归档。就像人类的选择性记忆。

**一句话**：「搜索一下我之前的记忆」「记忆树状态」「整理记忆」——剩下的交给它。

---

## 为什么需要它？

OpenClaw 用久了，`MEMORY.md` 会膨胀：

| 问题 | 后果 |
|------|------|
| 过期信息不清理 | 每轮对话白烧 token |
| 重要知识淹没 | 龙虾"记性变差" |
| 手动维护太累 | 文件越来越乱 |

记忆树用**置信度机制**解决这个问题——用进废退，自然衰减。

---

## 核心功能

### 🔍 语义搜索
一句话：「搜索一下关于 xxx 的记忆」

智能理解语义，不只是关键词匹配。搜索命中自动提升记忆的"健康值"。

### 🌳 健康可视化
一句话：「记忆树什么状态」

查看记忆分布：多少活跃、多少衰减中、多少已归档。

### 🍁 自动衰减
一句话：「整理一下记忆」

置信度低的记忆自动归档。归档不是删除——精华保留，随时可找回。

### 📌 优先级保护
给核心知识标记 `[P0]`，永远不会衰减。

---

## 置信度机制

每条记忆有健康值（0.0–1.0）：

| 阶段 | Emoji | 分数 | 含义 |
|------|-------|------|------|
| 新芽 | 🌱 | 0.7 | 新知识 |
| 翠绿 | 🌿 | ≥0.8 | 常用，茁壮 |
| 枯黄 | 🍂 | 0.5–0.8 | 偶尔用，衰减中 |
| 凋零 | 🍁 | 0.3–0.5 | 很少用，即将归档 |
| 落土 | 🪨 | <0.3 | 已归档 |

**变化规则**：
- 被搜索命中：+0.03
- 每天未访问：-0.004 到 -0.008
- P0 标签：永不衰减

---

## 搜索后端

自动检测，优先级：

1. **Ollama**（本地，免费，隐私）— 推荐
2. **智谱 API**（云端，极低费用）
3. **OpenAI 兼容**（云端）
4. **关键词 fallback**（零依赖）

用 Ollama 后端时，记忆数据永不离开你的机器。

## 安装

```bash
# 从 ClawHub 安装
openclaw skill install memory-tree

# 或从 GitHub 克隆
git clone https://github.com/Masongmx/memory-tree.git
cp -r memory-tree/skill ~/.openclaw/workspace/skills/memory-tree
```

初始化一句话：「帮我初始化记忆树」——自动创建定时任务，之后免维护。

---

## 使用方式

**直接说人话**：

| 你说 | 它做 |
|------|------|
| 「搜索我之前说的 xxx」 | 语义搜索记忆 |
| 「记忆树状态」 | 输出健康报告 |
| 「整理记忆」 | 执行衰减，归档低置信记忆 |
| 「把这条标记为 P0」 | 永久保护，不衰减 |

## 技术规格

- Python 3.8+，零外部依赖

- 支持 macOS / Linux / WSL
