# 🌳 记忆树

让龙虾拥有人类般的记忆——记住重要的，忘记过期的。

**装上就自动运行**：每天衰减、每周归档，无需任何操作。

---

## 为什么做这个项目？

灵感来自 [Memory-Like-A-Tree](https://github.com/loryoncloud/Memory-Like-A-Tree)，但原项目有几个问题：

| 原项目的问题 | 我们的解法 |
|-------------|-----------|
| 需要手动配置环境变量、路径 | **零配置**：自动检测 OpenClaw 环境 |
| 只有关键词搜索，搜"辣的"找不到"火锅" | **语义搜索**：多后端自动切换 |
| 独立工具，要自己跑脚本 | **OpenClaw 技能**：一句话调用 |
| 10+ 个脚本，架构复杂 | **单文件**：300 行 Python |
| 需要 Obsidian 才能查看归档 | **内置归档**：精华自动提取到 JSON |

---

## 我们的特点

| 特点 | 说明 |
|------|------|
| **零配置** | 自动检测环境，不用改配置文件 |
| **少依赖** | Python 标准库，Ollama 可选 |
| **小白友好** | 一句话初始化，之后全自动 |

---

## 核心原理：置信度

每条记忆有个"健康值"（0.0–1.0）：

| 状态 | Emoji | 含义 |
|------|-------|------|
| 活跃 | 🌿 | 经常用，置信度 ≥0.8 |
| 衰减中 | 🍂 | 偶尔用，0.5–0.8 |
| 已归档 | 🪨 | 不用了，<0.3 |

**变化规则**：
- 被搜索命中：+0.03（用进）
- 每天未访问：-0.004 到 -0.008（废退）
- 标记 [P0]：永不衰减

## 你只需要知道两件事

### 1. 想回忆某事？
说：「帮我回忆一下 xxx」「搜一下我之前说的 xxx」

语义搜索，能理解"辣的"和"火锅"相关。

### 2. 想让龙虾永远记住某件事？
说：「记住这个」「这个很重要」「别忘了我喜欢吃辣」

龙虾会把它设为永久记忆，永远不会忘。

---

## 其他的全自动

装好初始化后：
- 每天凌晨 3 点自动衰减
- 每周自动归档死亡记忆
- 搜索引擎自动检测最佳后端

## 安装

```bash
git clone https://github.com/Masongmx/memory-tree.git
cp -r memory-tree/skill ~/.openclaw/workspace/skills/memory-tree
```

初始化：「帮我初始化记忆树」
